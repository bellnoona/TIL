# Contents Model (컨텐츠 모델)

> HTML5 이전에는 HTML 태그 요소를 inline(인라인) 요소와 block(블록) 요소 두가지로 분류해왔다.<br>
> HTML5 이후부터는 요소의 특성에 따라 카테고리를 정의하여, 요소별로 공통 특성을 공유하는 것끼리 그룹화한다.
> 태그의 특성에 따라 7개의 카테고리로 분류하였고
